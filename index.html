Step 1 — Open your Google Sheet

  1. Go to sheets.google.com and open your spreadsheet (or create a blank one)
  2. In the menu bar click Extensions
  3. Click Apps Script — a new tab opens with a code editor
💡 Everything runs inside your own Google account. InvoiceLens never sees your credentials.

Step 2 — Paste the script

  1. In the Apps Script editor press Ctrl+A to select all existing code
  2. Click 📋 Copy Script below, then paste with Ctrl+V
  3. Press Ctrl+S to save (name the project anything)

Step 3 — Deploy as a Web App

  1. Click the blue Deploy button (top right of the editor)
  2. Choose New deployment
  3. Click the ⚙ gear icon next to "Select type" → pick Web app
  4. Set: Execute as: Me  ·  Who has access: Anyone
  5. Click Deploy → Authorize access → sign in → Allow
  6. Copy the Web app URL (starts with https://script.google.com/macros/s/)
⚠ Keep this URL private — anyone with it can add rows to your sheet.
🔄 Updated the script? You must create a New deployment each time — not "Manage deployments".

Step 4 — Paste your Web App URL

Must contain script.google.com/macros/s/ and end with /exec
✓ URL saved! InvoiceLens is now linked to your Google Sheet.
